# Protocol Decoder

A serial communication protocol decoder and analyzer for decoding, visualizing, and managing various communication protocols.

## Screenshots

[Screenshot placeholder - Protocol decoder interface with data visualization]

## Features

- **Multi-Protocol Support** - Decode multiple communication protocols
- **Real-Time Data Decoding** - Process and decode incoming data streams
- **Protocol Visualization** - Visual representation of protocol data structure
- **Custom Protocol Definitions** - Create and define custom protocols
- **Data Import/Export** - Import captured data and export decoded results

## Tech Stack

- HTML5
- JavaScript
- Progressive Web App (PWA)

## Getting Started

### Quick Start

1. **Online Demo**: Visit [Protocol Decoder Live](https://arnoutzw.github.io/protocol-decoder-app/)
2. **Local Setup**: Clone the repository and open `index.html` in your web browser

```bash
git clone https://github.com/arnoutzw/protocol-decoder-app.git
cd protocol-decoder-app
# Open index.html in your browser
```

### Installation

This is a PWA that works entirely in your browser with no server or dependencies required. For best experience:
- Install as a PWA on your device for offline access
- Use a modern web browser (Chrome, Firefox, Safari, Edge)

## Usage

1. **Select Protocol** - Choose a protocol or create a custom one
2. **Input Data** - Paste or import raw serial data
3. **Decode** - Analyze and decode the protocol data
4. **Visualize** - View decoded data in structured format
5. **Export** - Save decoded results for further analysis
